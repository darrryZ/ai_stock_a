# 微信公众号配置指南

## 1. 前置条件

- 已注册微信公众号（订阅号即可）
- 项目已部署到 Vercel（获取域名）

## 2. 设置环境变量

在 Vercel 项目设置 → Environment Variables 中添加：

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `WECHAT_TOKEN` | 微信服务器验证 Token | `your_custom_token_here` |

## 3. 微信公众平台配置

登录 [微信公众平台](https://mp.weixin.qq.com/) → 设置与开发 → 基本配置：

1. **服务器配置** → 修改配置
   - **URL**: `https://your-domain.vercel.app/api/wechat`
   - **Token**: 与环境变量 `WECHAT_TOKEN` 保持一致
   - **EncodingAESKey**: 随机生成即可
   - **消息加解密方式**: 明文模式

2. 点击 **提交** 验证

3. 验证成功后 **启用** 服务器配置

## 4. 使用方式

关注公众号后，直接发送 6 位股票/基金代码：

```
600519     → 贵州茅台分析
000001     → 平安银行分析
159915     → 创业板ETF分析
510300     → 沪深300ETF分析
```

## 6. 注意事项

- 微信订阅号被动回复有 **5秒超时限制**
- 如遇超时，可能需要优化 API 响应速度或使用客服消息接口
- 当前为明文模式，生产环境建议切换为安全模式
